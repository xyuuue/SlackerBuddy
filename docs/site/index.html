SlackerBuddy

一只会陪你偷懒，也会提醒你休息的 Mac 小猫。

FuFu 是 SlackerBuddy 的暹罗猫桌面宠物。它会眨眼、挥手、跳跃、等待、奔跑，在你工作的时候陪在屏幕边上，并按你设置的节奏提醒休息和喝水。

下载 SlackerBuddy.dmg 适用于 macOS 14 或更新版本
FuFu 会在你专注时安静待机，也会在提醒时间到时挥手出现。

认识 FuFu

像桌面伙伴一样陪伴

小猫可以拖动到你喜欢的位置，也能调整大小，安静地停在工作区旁边。

有反应的小动作

点击、拖拽、提醒和自动动作都会触发不同反馈，包括审视、跳跃、等待和奔跑。

休息和喝水提醒

休息间隔、喝水间隔、气泡显示时长都可以自定义，也可以开启低干扰模式。

下载与安装

下载 DMG 后，把 SlackerBuddy 拖到 Applications。第一次打开未公证版本时，macOS 可能要求你在系统设置里允许打开。

  1. 1

    点击页面上方的下载按钮，保存 SlackerBuddy.dmg。

  2. 2

    打开 DMG，把 SlackerBuddy.app 拖入 Applications。

  3. 3

    从 Applications 启动 SlackerBuddy，在菜单栏里打开设置并选择你喜欢的提醒节奏。
